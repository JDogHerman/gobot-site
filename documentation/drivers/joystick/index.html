Drivers

Cylon has a extensible system for connecting to hardware devices.

Generic Joystick

Driver

Allows user to interact with gobot devices using any joystick or game controller that is compatible with SDL.

How to establish a connection and setup the driver

require 'gobot'
connection :joystick, :adaptor => :joystick
device :controller, :driver => :joystick, :connection => :joystick, :interval => 0.1

Public driver events

joystick

Get's triggered every "interval" amount of time if a joystick event occured, you can access values x, y, s.

button

Get's triggered every "interval" amount of time if the button was pressed, you can access the value of the pressed button.

How to use events and call them on your drivers

Pretty simple, when the connection and the driver have been setup (as explained above) you can call events on them as follows:

on joystick, :joystick => proc { |caller, value|
  puts 'joystick ' + value[:s].to_s, value[:x], value[:y]
}

How to use

Example receiving button events from a generic joystick.

require 'gobot'

connection :joystick, :adaptor => :joystick
device :joystick, :driver => :joystick, :connection => :joystick, :interval => 0.1

work do
  on joystick, :joystick => proc { |caller, value|
    puts 'joystick ' + value[:s].to_s, value[:x], value[:y]
  }
  on joystick, :button => proc { |*value|
    puts 'button ' + value[1]
  }
end
