What is Gobot?

Gobot is a framework for robotics, physical computing, and the Internet of Things, written in the Go programming language. It provides a simple, yet powerful way to create solutions that incorporate multiple, different hardware devices at the same time.

Want to use Node.js on robots? Check out our sister project CylonJS (http://cylonjs.com)

Want to use Ruby on robots? Check out our sister project Artoo (http://artoo.io)
